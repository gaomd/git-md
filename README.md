git-md v0.1.0
=============

A Markdown helper used directly inside Git.

COMMANDS
--------

    help            show the help
    list            list files with .md suffix, recursively
    edit <path>     edit file at the given path, using the $EDITOR

EXAMPLES
--------

    $ git md list
    $ git md edit blog/hello-world.md

INSTALLATION
------------

    Drop `git-md` executable to $PATH
